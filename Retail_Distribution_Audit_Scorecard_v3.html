Google Sheets Integration
One-time setup. Once connected, every submitted audit automatically appears as a new row in your separate Retail Distribution Google Sheet — completely independent from the TC Direct Store Audit sheet. You will need to create a new sheet and deploy a new Apps Script for this scorecard.
1
Create a new Google Sheet (separate from TC Direct) and add these headers in Row 1:
Timestamp | Country | Store | Brand | Tier | CM | Auditor | Date | Type | Score A | Score B | Score C | Score D | Score E | Score F | Overall % | Grade | Items Passed | Total Items | Failed Items | Actions
2
In that new sheet, open Extensions → Apps Script, delete any existing code, paste the script below, then Save. Do not reuse the TC Direct Apps Script — this needs its own deployment.
function doPost(e) { var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet(); var data = JSON.parse(e.postData.contents); sheet.appendRow([ new Date().toISOString(), data.country, data.store, data.brand, data.tier, data.cm, data.auditor, data.date, data.type, data.scoreA, data.scoreB, data.scoreC, data.scoreD, data.scoreE, data.scoreF, data.overall, data.grade, data.passed, data.total, data.failedItems, data.actions ]); return ContentService.createTextOutput(JSON.stringify({status:"success"})) .setMimeType(ContentService.MimeType.JSON); } function doGet(e) { return ContentService.createTextOutput("Retail Audit endpoint is live.") .setMimeType(ContentService.MimeType.TEXT); }
3
Deploy as Web App: Click Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone → Deploy → Copy the Web App URL.
4
Paste your Web App URL below — saved in your browser, one-time only.

Audit results are stored locally in your browser so the Dashboard works offline. Submitted audits also send to Google Sheets if configured above.
