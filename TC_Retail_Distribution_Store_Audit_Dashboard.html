Google Drive Auto-Status (Advanced)
To auto-detect when CMs upload photos to Google Drive, set up a Google Apps Script that:
1. Scans a shared Drive folder for files uploaded in the past 7 days
2. Matches file names to store codes
3. Updates a Google Sheet with submission status
4. This dashboard reads from that Sheet URL to show live status

This requires a shared Google Drive folder where CMs upload photos using the naming convention: [StoreCode]_[Section]_[YYYYMMDD]
